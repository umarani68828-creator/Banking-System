# Banking Management System (SQL)

This project is a relational database management system for a banking application built using SQL. It demonstrates database design, normalization, and query operations by implementing four interconnected tables:

Customers – Stores customer information.
Accounts – Stores account details and balances.
Transactions – Records deposits and withdrawals.
Loans – Maintains customer loan information.
# Features
Database creation using SQL DDL commands
Primary Key and Foreign Key relationships
50 sample records inserted into each table
Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
GROUP BY queries for analysis and reporting
Multi-table JOIN operations
Sorting and filtering queries (ORDER BY, WHERE, LIMIT)
